## Star Wars

**First step:**
git clone https://github.com/tiagocharo/star-wars.git

**Second step:**
cd star-wars

**Third step:**
npm install

**Fourth step:**
npm start

**Fifth step:**
Open http://localhost:3000 to view it in the browser.
